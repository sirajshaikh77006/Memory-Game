Rules

How to Play:

  1. Start the Game
    • Press any key to begin the game.
    • The game will display a sequence of colors, starting with one color.
  2. Memorize the Sequence
    • The game will highlight a button (e.g., red, blue, green, or yellow).
    • A sound will play corresponding to the color.
    • Each round, a new color is added to the sequence.
  3. Repeat the Pattern
    • Click the buttons in the exact order shown by the game.
    • If you correctly follow the pattern, you advance to the next level.
    • The sequence gets one step longer each round.
  4. Making a Mistake
    • If you click the wrong button, the game will:
      • Play a "wrong" sound
      • Flash the screen with a red "game-over" effect
      • Display "Game Over! Press any key to restart."
    • Your score resets to 0, and you have to start again.
